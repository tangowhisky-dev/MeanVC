**Milestone:** M001

## S04 UAT

1. Launch app → Library page → click "+ New" → enter name → profile appears in list
2. Click profile → "Add Audio" → select wav → see "Extracting…" button state → "ready" status in list
3. Select audio → "Set as Default ★" → star appears
4. "Export" → saves zip file
5. "Import" → restores profile from zip
6. "Delete" → profile and disk files removed
